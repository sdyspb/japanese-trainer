# Японский тренажёр

Интерактивное веб-приложение для запоминания японских слов методом «верно/неверно» с таймером и озвучиванием.

## Использование

1. Откройте [GitHub Pages](https://USER.github.io/japanese-trainer/).
2. Нажмите одну из кнопок этапа (1, 2 или 3).
